## How to build it using CMake

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Admaaxd/Terrain-Generation.git
   cd Terrain-Generation

2. **Build the project**
   ```bash
   mkdir build
   cd build
   cmake -DCMAKE_BUILD_TYPE=Debug ..
   cmake --build .
   
3. **Run the program**
   ```bash
   cd Debug
   Terrain.exe
